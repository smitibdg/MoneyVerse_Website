MoneyVerse is a responsive web application designed to simplify personal finance management. It helps users track their income and expenses by automatically categorizing them according to the popular **50/30/20 rule** (50% Needs, 30% Wants, 20% Savings).

Built entirely with **Vanilla JavaScript**, this project focuses on a clean user interface and client-side data persistence.

## ğŸš€ Features
* **Smart Budgeting:** Automatically calculates how much you should spend on Needs, Wants, and Savings based on your income.
* **Expense Tracking:** Log daily expenses and see them deducted from the relevant categories in real-time.
* **Data Persistence:** Uses `localStorage` to save your profile, budget, and expenses. *No data is lost when you refresh the page!*
* **Financial Literacy:** Includes interactive quizzes to test your knowledge on finance basics.
* **Responsive Design:** Works seamlessly on desktops, tablets, and mobile phones.

## ğŸ“Š The 50/30/20 Rule Explained
This app is built on the principle that a balanced budget should be divided as follows:
* **50% Needs:** Essential expenses (Rent, Groceries, Bills).
* **30% Wants:** Discretionary spending (Dining out, Entertainment, Hobbies).
* **20% Savings:** Future goals (Emergency fund, Investments).
